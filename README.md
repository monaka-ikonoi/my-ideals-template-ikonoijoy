# My Ideals 用 ＝LOVE / ≠ME / ≒JOY テンプレート

[My Ideals](https://github.com/monaka-ikonoi/my-ideals) で使える、イコノイジョイの生写真管理用テンプレートです。

## テンプレート一覧

下のリンクを**右クリック（または長押し）してURLをコピー**してください。

| グループ | テンプレートURL | 状態 |
| :--- | :--- | :--- |
| **＝LOVE** | (準備中) | 🚧 |
| **≠ME** | [not-equal-me.json](https://my-ideals-ikonoijoy-prod.chocomonaka.dev/templates/not-equal-me.json) | ✅  |
| **≒JOY** | [nearly-equal-joy.json](https://my-ideals-ikonoijoy-prod.chocomonaka.dev/templates/nearly-equal-joy.json) | ✅  |


## 使い方

1. 上記のリンクを**右クリックや長押しでコピー**します。
2. My Ideals を開き、以下のメニューから **「新規プロファイル」** を選択します。
  * **PCの場合**：右上のドロップダウンリストをクリック
  * **スマホの場合**：右上の三本線（ハンバーガーメニュー）をクリック
3. コピーした URL を貼り付けて追加すれば完了です！
