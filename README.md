## 使い方

1. 上記のリンクを**右クリックや長押しでコピー**します。
2. My Ideals を開き、以下のメニューから **「新規プロファイル」** を選択します。
  * **PCの場合**：右上のドロップダウンリストをクリック
  * **スマホの場合**：右上の三本線（ハンバーガーメニュー）をクリック
3. コピーした URL を貼り付けて追加すれば完了です！
